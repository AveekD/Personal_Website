Wikipedia Graph Network

The Wikipedia Game is a game that basically pits random players against one another to find the fastest link to a specified Wikipedia article from an arbitrary starting point. I built a Chrome extension capable of finding the shortest route between these chosen articles and beat any human player ;). I built a chrome extension that used a set of API calls to a database with all of the English Wikipedia Links. This database effectively stored Wikipedia as a Graph Network using Stanford's SNAP project and then utilized A-star's algorithm to find the shortest route.
